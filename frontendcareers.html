Front-End Career

Front-End developers are responsible for the content that the user or client. They can control the way the webpage looks and feels. Some careers that a front-end developer might opt for would be, Working remotely, working with a individual contractor or working with a large company. Many people like to be there own boss so they like to work as an individual contractor. Some people like more structure so they choose to work for a large company. The bigger the company, the bigger the salary. The main way to get started would be to build your portfolio and showcase it on GitHub.
